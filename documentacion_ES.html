1. Auditoria Wifi


Untangle

'Untangle', lo que hace este sistema operativo es implementar seguridad en red y monitorizar la red.
Una vez descargado el archivo .ISO, usamos el programa 'Balena Etcher' y con un USB lo conectamos a un PC sin OS.

Enlace para descargar Untangle


Kali Linux


En Kali Linux,lo primero que hicimos fue instalar el sistema operativo 'Kali Linux' en 2 portatiles.
Después abrimos un programa que viene integrado con este OS llamado 'Wifite'.

Como se puede observar en la imagen, lo que nos dice al intentar atacar con este programa a nuestro wifi, dice que es imposible de atacar, ya que en la configuracion del router habilitamos el firewall y el anti-DDOS.




Raid

Para hacer el raid fue muy simple, hicimos una máquina virtual en VirtualBox. Dentro de esta máquina, metimos 3 discos para hacer el Raid 1.
El Raid 1 lo hicimos gracias a la documentación de Iker.
